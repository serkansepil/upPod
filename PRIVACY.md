# Privacy

UpPod is designed as a local-first Mac app.

## Data Used

UpPod uses AirPods motion data while the app is running to estimate head tilt and posture strain.

## Data Stored

UpPod stores the following locally on the Mac:

- calibration data
- sensitivity setting
- daily posture summaries
- exercise session summaries

The local file is:

```text
~/Library/Application Support/uppod/state.json
```

## Data Sharing

UpPod does not send posture, exercise, calibration, or motion data to a server.

## Permissions

macOS may ask for Motion & Fitness permission. This permission is required for AirPods motion tracking.
